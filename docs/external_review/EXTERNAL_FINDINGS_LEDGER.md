# External Findings Ledger

## Severity Definitions

- INFO: Informational note or documentation observation.
- LOW: Minor issue with limited reproduction impact.
- MEDIUM: Issue that affects clarity, setup, or expected-output comparison.
- HIGH: Issue that blocks meaningful local reproduction.
- CRITICAL: Issue that invalidates a core reproduced claim.
