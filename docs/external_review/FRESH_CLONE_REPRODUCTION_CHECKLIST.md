- [ ] I cloned the repository from the public GitHub URL.
- [ ] I recorded the commit hash.
- [ ] I recorded the tag or branch used.
- [ ] I created a clean Python virtual environment.
- [ ] I installed the package locally.
- [ ] I ran the full test suite.
- [ ] I ran the public demo.
- [ ] I ran release package validation.
- [ ] I ran external review packet validation.
- [ ] I compared output against expected outputs.
- [ ] I inspected the generated report or sample report.
- [ ] I inspected the generated viewer or sample viewer files.
- [ ] I confirmed the demo did not require real deployment.
- [ ] I confirmed the demo did not require GitHub writes.
- [ ] I confirmed the demo did not expose secrets.
- [ ] I recorded any failures in the failure log.
- [ ] I completed the reviewer attestation form.
